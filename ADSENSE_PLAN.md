# AdSense Plan

The site can pursue Google AdSense after it has enough useful, original content and clear navigation.

## Current status

- Public URL: https://wognsdl2.github.io/ai-dev-tool-stack/
- Static HTML site is live.
- About, Contact, and Privacy pages are present.
- Five article pages are present.
- Affiliate disclosure is visible.

## Human-only steps later

1. Create or log in to Google AdSense.
2. Add the site URL.
3. Copy the AdSense verification code.
4. Send the publisher/client code so it can be placed in the site.
5. After approval, add `ads.txt` with the exact publisher ID Google provides.

## Practical threshold before applying

Apply after the site has:

- 10 to 15 useful articles.
- A privacy policy and contact page.
- Search Console verification.
- At least some indexed pages.

Traffic is not the only approval factor. Thin or low-value content can be rejected even when a site has visitors.
