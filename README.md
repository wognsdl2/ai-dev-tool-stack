# AI Dev Tool Stack

Static affiliate/SEO seed site for AI coding tools and solo-dev launch tools.

## What is ready

- Home page with comparison cards and disclosure.
- Three SEO seed articles.
- Generated hero image.
- Placeholder affiliate link map.
- Basic robots and sitemap files.

## Human-only setup

1. Pick a domain.
2. Deploy this folder to Cloudflare Pages, Netlify, Vercel, or GitHub Pages.
3. Apply to affiliate programs.
4. Replace links marked with `data-affiliate` after approval.
5. Update `sitemap.xml` and canonical URLs if a custom domain replaces GitHub Pages.

## First programs to apply to

- Namecheap affiliate program through Impact or CJ.
- PartnerStack or other networks for AI coding tools.
- Hosting provider affiliate program only after choosing the recommended hosting angle.

## Compliance notes

- Keep affiliate disclosure visible on every page with monetized links.
- Do not claim a tool is best only because it pays commission.
- Verify prices, plans, commissions, and payout rules before publishing exact numbers.
